Welcome to my dotfiles repo! This contains all my personal configurations and scripts to set up a fully riced and productive Arch Linux environment. It includes everything from terminal settings and launchers to keybinds and shell utilities.

## 📦 What’s Inside

This repository includes configuration files and setup for:

- 🐧 **Arch Linux** (btw)
- 🖥️ **Neofetch**, **Fastfetch**
- 🧠 **Ghostty**, **Alacritty**
- 🎯 **Rofi**, **Ulauncher**
- 🧰 Shell scripts & automation
- 🔧 Custom KDE keybinds
- 🖼️ Wallpapers
- 🗂️ Stow for dotfile management

---

## ⚙️ Installation

To install all configs and set up the environment:

```bash
git clone https://github.com/abhimangs/dotfiles.git
cd dotfiles
chmod +x install.sh
./install.sh
```

🛠 Install Dependencies
Run this script before installing the dotfiles to make sure all required packages are installed:

```bash
sudo pacman -Syu --needed git stow
```

## 🧪 Example: Dotfiles Installation Menu

When you run the install process, you’ll see this interactive menu:

```bash
==================================
  🔧 Dotfiles Configuration Menu  
==================================

1) All  
2) Bash  
3) Fastfetch  
4) Ghostty  
5) KDE Keybinds  
6) Neofetch  
7) Pacman  
8) Rofi  
9) Ulauncher  
10) Wallpaper  
11) Exit  

==================================

Enter numbers (e.g., 2 5 9): 2 6 10
```

In this example:  
- `2 → Bash`  
- `6 → Neofetch`  
- `10 → Wallpaper`  

These selections mean that **Bash**, **Neofetch**, and **Wallpaper** will be installed with custom configurations and dotfiles from this repository. Each option sets up the respective tool or component with personalized settings tailored to enhance your workflow and aesthetics.  

To install everything with all custom configurations, simply enter `1`.
